# Calculator Application

This is a simple calculator application built with TypeScript. It provides basic arithmetic operations including addition, subtraction, multiplication, and division.

## Project Structure

```
calculator-app
├── src
│   ├── app.ts               # Entry point of the application
│   ├── components
│   │   └── Calculator.ts    # Calculator component with arithmetic methods
│   └── types
│       └── index.ts        # Type definitions for calculator operations
├── package.json             # npm configuration file
├── tsconfig.json            # TypeScript configuration file
└── README.md                # Project documentation
```

## Features

- Perform basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- User input handling
- Result display

## Installation

To install the necessary dependencies, run:

```
npm install
```

## Usage

To start the application, run:

```
npm start
```
